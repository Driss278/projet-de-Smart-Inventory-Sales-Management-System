# Smart Inventory & Sales Management System
Smart Inventory & Sales Management est un mini-ERP pour entreprise: gestion produits, clients et commandes avec OOP, exceptions métiers, DAO + MySQL, interface web Django (CRUD, historique). Export CSV puis analyse Pandas/NumPy (CA mensuel, top produits, valeur stock, panier moyen, fréquence clients) avec graphiques, KPI et dashboard de suivi pro..
